# ⚽ World Cup 2026 Predictor

A data-driven football match predictor using ELO ratings, squad strength,
player form scores, and a Gradient Boosting classifier.

## Quickstart

```bash
pip install -r requirements.txt

# 1. Drop Kaggle CSVs into data/raw/
#    results.csv, goalscorers.csv, shootouts.csv,
#    fifa_players.csv, rankings.csv, player_ratings.csv

# 2. Run the full pipeline
python pipeline.py

# 3. Launch the web app
streamlit run app/app.py
```

## Project Layout

```
world_cup_predictor/
├── data/raw/          ← Kaggle files (never edit)
├── data/processed/    ← Generated by pipeline
├── data/external/     ← Small JSON reference files
├── notebooks/         ← Exploration notebooks
├── src/               ← All production Python code
├── models/            ← Saved .pkl model files
├── app/               ← Streamlit web app
├── tests/             ← Unit tests
└── pipeline.py        ← Run everything in one command
```

## Expected Accuracy
| Model | Accuracy |
|-------|----------|
| Baseline (always home win) | ~45% |
| ELO only | ~58% |
| ELO + form | ~62% |
| Full feature set + GBM | ~65–68% |
| + Squad / player data | ~70%+ |
